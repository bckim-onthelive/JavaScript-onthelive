문제)
 로또 당첨 예상 번호를 만드는 프로그램을 만든다.
 사용자가 수동으로 입력된 번호를 화면에 표출 할 수도 있고,
 프로그램상 자동으로 번호를 만들어서 화면에 표출 할 수도 있다.

1. Math.Random() 을 이용해서 실제 로또 번호 1부터 45까지의 숫자 중 6개를 뽑아 [로또 당첨 번호]를 설정해둔다.

3. 사용자의 입력을 통해서 자동으로 추첨할 건지, 수동으로 추첨할 건지 선택한다.
4. 사용자가 뽑은 로또 번호가 테이블로 표현되며, 화면상 ui로 사용자가 확인 할 수 있어야 한다.
5. 위 과정이 완료되면, 로또 당첨 번호와 사용자의 번호를 비교하여 당첨 결과를 확인한다. 이때,
6. 1등 : 1000만원, 2등 20만원, 3등 10만원, 4등 5000원, 5등 1000원을 지급하며 [기존 로또 룰에서 금액 변경]
